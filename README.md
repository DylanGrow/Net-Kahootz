# NetPlus Pro - CompTIA Network+ Practice Platform

## Deployment to GitHub Pages

1. Create new GitHub repository
2. Upload these files to main branch:
   - index.html
   - robots.txt
   - sitemap.xml
3. Go to Settings > Pages
4. Deploy from main branch / root
5. Your site will be live at https://yourusername.github.io/repo-name/

## Setup Google Sheets

1. Create Google Sheet with tabs: Questions, GameState, Responses
2. Publish to web as CSV
3. Copy Sheet ID from URL
4. Open site, enter Sheet ID when prompted

## Files Included

- index.html - Main application (34KB, works offline)
- robots.txt - SEO crawler instructions
- sitemap.xml - Search engine sitemap

## Features

- Kahoot-style live quizzes
- Spaced repetition learning
- Offline support via Service Worker
- Fully accessible (WCAG 2.1 AA)
- Mobile responsive
- No backend required
