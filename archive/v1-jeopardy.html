How to Setup

  1. Create a Google Sheet with two tabs named exactly: Questions and State.
  2. Questions sheet — headers in row 1: Category, Value, Question, Answer
  3. State sheet — headers: key, value. Add rows: 
    key,value
current,
status,idle
lastUpdated,
  4. Share: File → Share → Anyone with the link → Viewer
  5. Publish: File → Publish to web → Entire document → Web page → Publish
  6. Copy Sheet ID from URL: https://docs.google.com/spreadsheets/d/<THIS_PART>/edit
  7. Host controls: As host, click a tile then click "Show Question". Copy the values into your State sheet:
    • Set current = Category|Value (e.g., Science|200)
    • Set status = question or answer or idle
    • Optional: set lastUpdated to now
