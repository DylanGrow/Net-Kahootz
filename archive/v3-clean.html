Setup Guide

1. Google Sheet Structure

Create a Sheet with three tabs:

Questions: Question | A | B | C | D | Correct | Explanation | Domain

GameState: status | current | startTime | pin (add one row of headers, leave data empty)

Responses: timestamp | pin | player | questionIndex | answer | correct | timeMs | points

Share sheet: Anyone with link → Viewer.

2. Apps Script

Extensions → Apps Script → paste this, deploy as Web App (Anyone):

const SHEET_ID_DEFAULT = '';
function doPost(e){
  const data = JSON.parse(e.postData.contents);
  const ss = SpreadsheetApp.openById(data.sheetId || SHEET_ID_DEFAULT);
  if(data.action === 'hostUpdate'){
    const sh = ss.getSheetByName('GameState');
    sh.getRange(2,1,1,4).setValues([[data.status, data.questionIndex, data.startTime, data.pin]]);
    return ContentService.createTextOutput('ok');
  }
  if(data.action === 'submitAnswer'){
    const sh = ss.getSheetByName('Responses');
    sh.appendRow([new Date(), data.pin, data.player, data.questionIndex, data.answer, data.correct, data.timeMs, data.points]);
    return ContentService.createTextOutput('ok');
  }
}
function doGet(e){ return ContentService.createTextOutput('ok'); }

3. Use

  1. Host: open Settings → paste Apps Script URL and full Sheet ID
  2. Host: enter name, paste full Sheet ID in PIN field, select Host → Join
  3. Players: enter name, enter last 6 chars of Sheet ID, select Player → Join
  4. Host controls Start / Next / Reveal. Timer is 20s. Points: 200–1000 for correct.
